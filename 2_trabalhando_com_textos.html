Trabalhando com textos no HTML

A TAG H1 na pagina define o titulo da pagina sendo o nivel mais alto, o uso dela indica o assunto principal do conteudo e é um elemento importante para o SEO, facilitando os motores de busca.


A TAG P é a definição de paragrafo no HTML, onde podemos colocar um texto longo ou bloco de texto.


Esse texto esta em H2 para subtitulos no HTML.

A Tag H2 é um subtitulo em uma pagina, ajuda a organizar e melhorar a leitura para usuarios e mecanimos de busca.


Outras Tags que podem ser usadas.

A Tag em faz com que o texto fique enfatizado, visualmente ela fica semelhante a Italico.

A Tag i faz com que o textos fique no modo Italico.

A Tag mark faz com que algum texto se destaque

A Tag small deixa o texto pequeno desta forma, usando a Tag b para o texto negrito. Texto pequeno

A Tag dell define que um texto tenha sido cancelado.

A Tag ins definque um um texto tenha sido inserido, DESTA FORMA
